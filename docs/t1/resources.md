# Recursos

Vocês têm à disposição boilerplates para a infraestrutura, o shell para o frontend, para o microfrontend e para o microsserviço. Clone os repositórios abaixo no mesmo diretório e use `docker` para rodar a aplicação a partir do repositório `chave-infra`:

- [chave-infra](https://github.com/pucrs-sweii-2026-1-31/chave-infra)
- [chave-shell](https://github.com/pucrs-sweii-2026-1-31/chave-shell)
- [chave-ms-auth](https://github.com/pucrs-sweii-2026-1-31/chave-ms-auth)
- [chave-mfe-auth](https://github.com/pucrs-sweii-2026-1-31/chave-mfe-auth)

Para o ADR, vocês podem utilizar como exemplo o [ADR da aplicação inteira](../about/adr.md), focando neste caso no escopo desta entrega.
